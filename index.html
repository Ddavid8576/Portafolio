Mis trabajos

Construyendo un landing page

En este desafío validamos nuestros conocimientos de CSS trabajados en clase y en la guía. Para lograrlo, necesitarás implementar CSS utilizando archivos de hojas de estilo externas al documento HTML y de base para homogeneizar el diseño del sitio. Además, deberás estructurar el proyecto usando la carpeta assets y sus subcarpetas.

Iguana page

En este desafío validaremos nuestros conocimientos de posicionamiento en CSS. Para lograrlo, necesitarás diferenciar los valores asociados a display (inline, block, inline-block), utilizar floats para posicionar texto e imágenes, y resolver el problema de colapso de floats utilizando un clearfix.

Cuppon

En este desafío validamos nuestros conocimientos de Bootstrap. Para lograrlo, necesitarás implementar el sistema de grillas de Bootstrap, distribuyendo el contenido adecuadamente, para hacer una página web adaptable a distintos dispositivos, utilizando la documentación disponible.
